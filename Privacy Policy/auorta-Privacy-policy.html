Privacy Policy

Effective Date: April 26, 2025

ASK Productions built the Auorta Learning app as a free educational platform for competitive exam aspirants. This SERVICE is provided at no cost and is intended for use as-is.

Disclaimer: Auorta Learning is a private educational platform and is not affiliated with or endorsed by any government entity. All materials are for educational purposes only and do not represent official government services or institutions.

Information Collection and Use

The app collects limited personal information such as:

  • User name and chosen exam (for profile personalization)
  • Profile image (stored locally only)
  • Email address (provided during profile setup)
  • Phone number (optional, if the user provides it for support or communication)
  • Study progress, syllabus completion, and mock test results (stored offline in the app's database)

No sensitive personal data such as precise location is collected.

File Uploads and Shared Data

Users can voluntarily upload educational files (e.g., notes or PDFs) through the app. When doing so, the following data is collected and shared with our internal Telegram system:

  • The file itself
  • Your name (entered in the app)
  • Your email address

This data is used only to review the content before publishing it on the app. We do not share this data with any third party except Telegram, which may store it as per their own privacy policy. We recommend avoiding uploading personally sensitive data in any file.

Data Shared: Yes, with Telegram (for file submission)

Data Collected: Yes, voluntarily provided file and contact info

Third-Party Services

The app uses the following third-party tools and services, which may collect anonymous data for analytics and technical improvements:

  • Google Firebase Analytics: Collects anonymized usage data to help improve user experience. No personally identifiable information is included in these analytics.
  • Firebase Cloud Messaging: Used to deliver optional push notifications.
  • MPAndroidChart: Used for visualizing data such as mock test performance. Does not collect personal data.
  • Retrofit and OkHttp: Used to securely load content from the internet. They do not collect user information by default.
  • Telegram Bots: Used to receive user-uploaded educational files. Limited personal info is shared voluntarily as detailed in the next section.

Permissions Used

  • INTERNET – For loading content and communicating with servers
  • POST_NOTIFICATIONS – To notify users about study sessions (Future Updates)
  • SCHEDULE_EXACT_ALARM – For accurate study session alerts (Future Updates)
  • RECEIVE_BOOT_COMPLETED – (Added by Firebase/WorkManager but not actively used)
  • WAKE_LOCK – To wake the device for scheduled notifications (Future Updates)
  • ACCESS_NETWORK_STATE – To check network status
  • FOREGROUND_SERVICE – To show timely notifications
  • AD_ID – Used if ads are enabled in future versions

Data Security

All user data, such as profile information, progress tracking, and syllabus completion status, are stored locally on your device. Uploaded files, along with your name and email address, are securely transmitted to our Telegram bot for internal review purposes. While we do not intentionally share this information with any third parties, please note that uploads are transmitted via Telegram, and are therefore subject to Telegram’s data policies and storage practices.

All communications are encrypted in transit using HTTPS or secure Telegram API endpoints.

Data Deletion

All user data stored locally on your device is automatically deleted if you uninstall the app.

If you have uploaded a file and would like it removed from our system, you may contact us via email. Please provide your name and email used during the upload. We will delete the file promptly if it is still within our review system.

Email: learningauorta@gmail.com

User Control and Opt-Out

You can choose not to use the file upload feature if you are uncomfortable sharing information. For Firebase Analytics, if you wish to opt out of anonymized analytics tracking, you can do so in your device settings (by disabling ad personalization or analytics permissions).

Children's Privacy

This app is not intended for use by individuals under the age of 13. We do not knowingly collect personally identifiable information from children. If we discover that a child under 13 has provided us with personal information, we will delete it immediately.

Changes to This Policy

This Privacy Policy may be updated from time to time. Users are advised to review this page periodically for any changes. Changes will be posted on this page and will become effective immediately after they are posted.

Contact Us

If you have any questions or suggestions about this Privacy Policy, feel free to contact us at learningauorta@gmail.com.
